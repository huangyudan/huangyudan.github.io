搭建Hexo博客时遇到的一些问题

网上的教程很多，比方说我看的这一篇蛮详细的，比较适合小白。
http://blog.hjtxxx.com/2015/08/13/Hexo-%E4%B8%80-%EF%BC%9A%E5%9C%A8GitHub%E4%B8%8A%E6%90%AD%E5%BB%BA%E9%9D%99%E6%80%81%E5%8D%9A%E5%AE%A2/
http://www.cnblogs.com/zhcncn/p/4097881.html
按照这两个教程说的去结合着做，大概分为以下步骤：下载并安装软件，我的电脑是win8，git和nodejs版本如下：git 2.6.4
node2.4.4，新建一个文件夹Hexo，右击出现git bash here，或者直接打开Git bash，用cd命令转到Hexo文件夹所在位置，根据上述教程开始安装。

在复制SSH KEY 的过程中pbcopy < ~/.ssh/id_rsa.pub出错，显示无复制命令
于是我去stack over flow上查找了答案：
You can list all the public keys you have by doing:
$ ls ~/.ssh/*.pub
电脑会显示这个文件在哪里，手动去找一下，打开复制粘贴就好，一般是在c盘的USER文件夹下

另外，在部署时出错hexo deploy命令显示ERROR Deployer not found : github，可能是git版本的问题，普遍的解决方法是站点配置文件_config.yml最下方deploy的type改成git，然后运行下npm install hexo-deployer-git –save
再hexo g
hexo d
